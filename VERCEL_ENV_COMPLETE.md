## üéØ How to Use

1. Go to Vercel Dashboard ‚Üí Your Project ‚Üí Settings ‚Üí Environment Variables
2. For each line above:
   - Click "Add New"
   - Copy the KEY (left side of `=`)
   - Copy the VALUE (right side of `=`)
   - Select all environments
   - Click "Save"
3. After adding all, redeploy your project
